# inventorytracker

## backend

### 1) Install gradle
Mac:
```
brew install gradle
```
### 2) Run backend application
```
gradle bootRun
```



## frontend

### 1) Install npm
```
npm install
```

### 2) Compiles and hot-reloads for development
```
npm run serve
```

Now, visit http://localhost:8081/ to access the application.
